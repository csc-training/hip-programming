# Tracing kernels

Gather trace with `rocprofv3` and visualize the execution timeline 
of [saxpy exercise](../kernels/kernels-saxpy/). Use [perfetto](https://ui.perfetto.dev) at least for question 1.

1. How long does the kernel execution take?
2. How many bytes are copied to device? 
3. How many bytes are copied to host?
4. How long does the fastest memory copy take?
